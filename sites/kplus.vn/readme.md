# kplus.vn

https://www.kplus.vn/highlights/broadcast-schedule

### Download the guide

```sh
npm run grab -- --site=kplus.vn
```

### Update channel list

```sh
npm run channels:parse -- --config=./sites/kplus.vn/kplus.vn.config.js --output=./sites/kplus.vn/kplus.vn.channels.xml
```

### Test

```sh
npm test -- kplus.vn
```
