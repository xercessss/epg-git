# tv.yettel.hu

https://tv.yettel.hu/

### Download the guide

```sh
npm run grab -- --site=tv.yettel.hu
```

### Update channel list

```sh
npm run channels:parse -- --config=./sites/tv.yettel.hu/tv.yettel.hu.config.js --output=./sites/tv.yettel.hu/tv.yettel.hu.channels.xml
```

### Test

```sh
npm test -- tv.yettel.hu
```
